Make Virtual Enviroment: 
which python
pip list | grep pyshark

python3 -m venv .venv
source .venv/bin/activate
deactivate

Use system wide packages:
python3 -m venv .venv --system-site-packages
source .venv/bin/activate
sudo pip install pyshark --break-system-packages
